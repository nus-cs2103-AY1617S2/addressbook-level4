Command Format (with some examples) Alternate Formats Description
Add add NAME [r/REMARKS] [s/STARTDATE] [s/ENDDATE] [l/LOCATION] [t/TAG]...

Example:
add Meet John s/tomorrow 3pm e/tomorrow 5pm r/regarding project l/canteen t/work 		- Adds a task to KIT with specifications from the user.
Clear clear Clears all tasks.
Delete delete INDEX

Example:
delete 1		 Deletes the specified task.
Done done INDEX

Example:
done 1		 d INDEX Marks the specified task as done.
Edit edit INDEX NAME [r/REMARKS] [s/STARTDATE] [e/ENDDATE] [l/LOCATION] [t/TAG]...

Example:
edit 1 s/Friday morning e/friday afternoon		 Edits an existing task.
Exit exit Exits the program.
Find find KEYWORD...

Example:
find meet		 f KEYWORD... Finds tasks that contains at least one keyword.
Find Exact findexact KEYWORD...

Example:
findexact meet john		 fexact KEYWORD..., finde KEYWORD..., fe KEYWORD... Finds tasks that contains all keywords.
Get Event from Google Calendar getgoogle gg Get upcoming events from your Google Calendar and add them as tasks.
Help help [COMMANDNAME]

Example:
help add		 h, man, manual If COMMANDNAME not entered, brings up user guide in another window.
If COMMANDNAME entered, shows help message for that command only.
Help Format helpformat helpf, summary, hf, man, manual Shows a quick summary of the command formats
List list ls, l Lists all tasks in KIT.
List Done listdone ld Lists all completed tasks.
List Not Done listnotdone listundone, lnd Lists all ongoing tasks.
List Tag listtag TAGNAME

Example:
listtag work		 listag TAGNAME, ltag TAGNAME, lt TAGNAME Lists all tasks of a certain tag.
Load load PATHNAME

Example:
load C:/Users/User/Documents/kit.xml		 Loads data from the specified file.
Post Event to Google Calendar postgoogle INDEX

Example:
pg 1		 pg Posts the specified task to your Google Calendar.
Redo Redo Revert the last undo.
Save save PATHNAME

Example:
save C:/Users/User/Documents/kit.xml		 Saves the file to a specified location.
Undo undo uhoh Undo the previous command.
Undone undone INDEX

Example:
undone 1		 notdone INDEX, ud INDEX, nd INDEX Marks the specified task as not done.
